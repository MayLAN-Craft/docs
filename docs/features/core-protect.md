# Проверка на гриф

Вам кажется, что украли ваши вещи из сундуков? Или кто-то взорвал крипера рядом с базой и вы хотите найти виновника?

!!! tip "Доступность режима инспектора"
    На нашем сервере каждому игроку доступен режим инспектора.

- Чтобы включить режим инспектора, введите команду `/co i`.
- Кликните ЛКМ по блоку, чтобы узнать, какие блоки на этом месте были поставлены или сломаны.
- Показывается также время изменения блоков и никнеймы игроков (или имена сущностей), которые эти блоки изменили.
- Вся информация выводится в чат. 
- Кликните ПКМ по грани блока, чтобы узнать историю изменений блока, который находился по другую сторону от грани.
- Нажмите ПКМ по любому механизму (кнопке, рычагу, двери), чтобы узнать, кто и когда использовал его.
- Режим инспектора отключается повторным вводом команды `/co i`.

После нахождения виновника у вас есть два пути: договориться мирно без участия администрации, либо если это не удаётся, написать об этом администрации сервера. Игрок, занимающийся гриферством, будет наказан любым внутриигровым способом, включая бан.
