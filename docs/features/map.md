# Динамическая карта

На сервере установлена динамическая карта BlueMap. IP адрес: https://maylan.dynmap.xyz/
!!! warning "Внимание"
    Динамическая карта может быть недотупной ввиду блокировок Интернета.

## Что доступно на карте?
- Карты трёх ванильных миров в 2D, 3D и с видом от первого лица
- Маркеры
- Метки игроков
- Обновление местности в реальном времени
- Слежка за игроками
- Координаты по XYZ
- Ссылки на определённую координату

## Как занести свою базу или ферму на карту?
Вы можете обозначить свою постройку на карте. Для этого напишите админу в личные сообщения.
