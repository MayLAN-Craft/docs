# Позы
На сервере доступно несколько поз для игроков:

- `/sit` - сесть на блок
- `/crawl` - ползти по земле
- `/lay` - лежать на спине 
- `/bellyflop` - лежать на животе
- `/spin` - кружиться на блоке
- `/sit toggle` - включить/отключить возможность сидеть на ступеньках
