# Скины
!!! warning "Внимание"
    Для показа скинов игроков на сервере необходимо отключить встроенные скины в настройках лаунчера.

Выдать себе скин можно 2 командами:

- `/skin <ник_игрока>` - ставит такой же скин, как и у игрока на лицензии.

- `/skin clear` - сброс скина

Использование скинов, изображающих политических деятелей или скинов с половыми органами **запрещено**.
