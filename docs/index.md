# MayLAN Craft

MayLAN Craft - приватный ванильный сервер Minecraft. Работает на версии 1.21.11

![Спавн сервера](https://raw.githubusercontent.com/MayLAN-Craft/docs/refs/heads/main/images/index.jpg)

## Что нужно знать о сервере

<div class="grid cards" markdown>

-   :material-cube:{ .lg .middle } __Ванильный сервер__

    ---

    Никаких `/god`, `/fly` и других кринжовых команд

-   :material-pirate:{ .lg .middle } __Пиратский__

    ---

    Для игроков с лицензией доступен вход без пароля

-   :lucide-earth-lock:{ .lg .middle } __Без приватов__

    ---

    Используется CoreProtect

-   :lucide-brush-cleaning:{ .lg .middle } __Без вайпов__

    ---

    Производим плавное расширение мира

-   :material-cash-off:{ .lg .middle } __Без донатов__

    ---

    Пока что :smile:

-   :lucide-infinity:{ .lg .middle } __Вечный__

    ---

    Сервер не умрёт ни через месяц, ни через два, ни через год


</div>

## Как подать заявку на сервер?

Заявку можно подать через группу Telegram или сервер Discord. В закреплённом сообщении есть пример оформленной заявки. Ознакомьтесь с правилами сервера, при подаче заявки будут вопросы на знание правил.

[Подать заявку в Discord :fontawesome-brands-discord:](https://discord.gg/vW5V8bKhve){ .md-button .md-button--primary }
[Подать заявку в Telegram :fontawesome-brands-telegram:](https://t.me/+Isyto5xN1e00YTFi){ .md-button }



## Меня приняли, что дальше?
В ответе к вашей заявке будут инструкции по входу на сервер. Также, перед запуском игры рекомендую установить мод [Simple Voice Chat](https://modrepo.de/minecraft/voicechat/wiki/installation) (особенно если вы согласились на войс в заявке).
