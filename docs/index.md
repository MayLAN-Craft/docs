# MayLAN Craft

MayLAN Craft - приватный ванильный сервер Minecraft. Работает на версии 1.21.11

## Что нужно знать о сервере

- Полностью ванильный сервер. Никаких `/god`, `/fly` и других кринжовых команд.
- Пиратский. Для игроков с лицензией доступен вход без пароля.
- Без приватов. Используется CoreProtect
- Без вайпов. Производим плавное расширение мира.
- Без донатов. Пока что :)
- Вечный. Сервер не умрёт ни через месяц, ни через два, ни через год.
- Имеется войсчат.

## Как подать заявку на сервер?

Заявку можно подать через [группу Telegram](https://t.me/+Isyto5xN1e00YTFi) или [сервер Discord](https://discord.gg/vW5V8bKhve). В закреплённом сообщении есть пример оформленной заявки. Ознакомьтесь с правилами сервера, при подаче заявки будут вопросы на знание правил.

## Меня приняли, что дальше?
В ответе к вашей заявке будут инструкции по входу на сервер. Также, перед запуском игры рекомендую установить мод [Simple Voice Chat](https://modrepo.de/minecraft/voicechat/wiki/installation) (особенно если вы согласились на войс в заявке).
